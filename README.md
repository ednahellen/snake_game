## 📋 Sobre o Projeto

Landing page moderna e responsiva para apresentar o clássico **Snake Game** (Jogo da Cobrinha), com uma versão interativa do jogo intitulada **"Cobrinha no Jardim do Éden"**.

O projeto foi desenvolvido como parte de um exercício acadêmico para demonstrar habilidades em desenvolvimento web, integrando HTML, CSS e JavaScript com um design inspirado no universo retrô/arcade.

---

## 🎯 Objetivo

Criar uma experiência imersiva que apresente o Snake Game de forma atrativa, combinando:

- Uma landing page visualmente impressionante com temática retrô
- Informações sobre o jogo e seus recursos
- Uma versão jogável do Snake diretamente na página
- Design responsivo para todos os dispositivos
- Imagens geradas por inteligência artificial

## 🎮 Funcionalidades do Jogo

O jogo **"Cobrinha no Jardim do Éden"** traz uma experiência única com:

- 🐍 **Controle intuitivo** com as setas do teclado
- 🍎 **Sistema de pontuação** com feedback visual (+1)
- 🏆 **High score** salvo no navegador (LocalStorage)
- 🌳 **Obstáculos** (árvores que bloqueiam o caminho)
- 👨👩 **Personagens** (Adão e Eva - Game Over ao tocá-los)
- ⚡ **Dificuldade progressiva** (velocidade aumenta a cada 5 pontos)
- 🎵 **Efeitos sonoros** (comer fruta e colisão)
- 💀 **Game Over** com animação e overlay visual

## 🤖 Como o ChatGPT foi Utilizado

O ChatGPT foi uma ferramenta essencial em todas as etapas do desenvolvimento:

### 1. 📝 Planejamento e Estrutura
- Auxiliou na definição da estrutura HTML semântica
- Sugeriu a organização das seções da landing page
- Ajudou a planejar a integração do jogo com a página

### 2. 🎨 Estilização CSS
- Gerou estilos modernos com tema retrô/arcade
- Criou animações fluidas (reveal, hover, shake)
- Desenvolveu um layout totalmente responsivo
- Sugeriu paleta de cores harmoniosa (verde neon, preto, cinza)

### 3. ⚡ Funcionalidades JavaScript
- Implementou o menu hamburger responsivo
- Criou o sistema de rolagem suave
- Desenvolveu o modal para ampliar imagens
- Implementou o botão "Voltar ao topo"
- Criou animações ao rolar (Intersection Observer)

### 4. 🎮 Desenvolvimento do Jogo
- Estruturou a lógica do Snake Game
- Adicionou recursos extras (obstáculos, personagens, high score)
- Implementou efeitos sonoros com Web Audio API
- Criou sistema de dificuldade progressiva

### 5. 🖼️ Geração de Prompts para Imagens
- Criou prompts detalhados para gerar imagens com IA:
  - **Hero**: Cobra digital estilizada em ambiente neon futurista
  - **Gameplay**: Tela de jogo retrô com cobra pixelada
  - **Concept**: Arte inspirada em fliperama com estilo arcade

### 6. 🔧 Correções e Melhorias
- Revisou e corrigiu bugs no código
- Sugeriu otimizações de desempenho
- Ajudou a refinar a experiência do usuário
- Auxiliou na documentação do projeto

### 7. 📚 Documentação
- Estruturou o README completo
- Organizou as seções e informações
- Criou a lista de funcionalidades e tecnologias
